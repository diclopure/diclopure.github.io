# DicloPure website

موقع عربي ثابت وتفاعلي لمشروع التخرج **DicloPure**. يعمل مباشرة في المتصفح ولا يحتاج إلى خادم أو قاعدة بيانات.

## الموقع العام

https://mohammedaa-os.github.io/diclopure/

## التشغيل المحلي

افتح `index.html` مباشرة، أو شغّل خادمًا محليًا من داخل المجلد:

```bash
python3 -m http.server 8080
```

ثم افتح `http://localhost:8080`.

## النشر

يمكن رفع المجلد كاملًا كما هو إلى GitHub Pages أو Netlify أو Vercel أو أي استضافة ملفات ثابتة. يجب بقاء `index.html` و`styles.css` و`script.js` ومجلد `assets` معًا.

## ما يستطيع المستخدم فعله

- تغيير تركيز الديكلوفيناك وكتلة الكربون ومعدل التدفق وزمن التشغيل.
- مشاهدة منحنى الاختراق والنتائج الهندسية مباشرة.
- مقارنة منحنيات Yan وYoon–Nelson وThomas بصريًا.
- تنزيل تقرير CSV كامل بالمدخلات والنتائج ونقاط المنحنى.
- نسخ رابط يحتفظ بقيم السيناريو لمشاركته.

## ملاحظة علمية مهمة

معاملات المحاكي الحالية معايرة أولية من الجداول المحتفظ بها في تقرير المشروع. قبل استخدام الأداة في تصميم هندسي فعلي، يجب استبدال طبقة `parameters()` داخل `script.js` بمعاملات الانحدار غير الخطي النهائية والتحقق منها ببيانات مستقلة كما تنص منهجية المشروع.
